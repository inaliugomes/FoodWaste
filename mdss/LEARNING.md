# React — Tópicos para aprender

## Obrigatório (antes de começar)

1. **JSX** — sintaxe que mistura HTML com JavaScript
2. **Componentes** — funções que retornam JSX, base de tudo em React
3. **Props** — passar dados de componente pai para filho
4. **useState** — guardar e actualizar estado dentro de um componente
5. **useEffect** — executar código quando o componente carrega (ex: chamar API)

## Aprender fazendo (quando precisar)

- **React Router** — navegação entre páginas
- **Formulários controlados** — inputs ligados ao estado com onChange
- **Fetch / Axios** — chamadas HTTP à API
- **Lifting state up** — partilhar estado entre componentes
